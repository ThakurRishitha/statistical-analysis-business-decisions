# Statistical Analysis for Business Decisions

## Project Overview
This project applies statistical methods to support business decision-making using the Boston Housing dataset.

## Objectives
- Conduct T-Test hypothesis testing
- Perform Chi-Square analysis
- Use probability distributions for risk analysis
- Conduct A/B testing
- Calculate confidence intervals
- Calculate margin of error

## Tools Used
- Python
- Pandas
- NumPy
- SciPy
- Matplotlib

## Key Findings

### T-Test
A significant difference was found in house prices between river-side and non-river-side properties.

### Chi-Square Test
River location was significantly associated with high house prices.

### Risk Analysis
The probability of a house price exceeding 30 was approximately 20.84%.

### Confidence Interval
95% Confidence Interval: 21.73 to 23.33

### Margin of Error
±0.80

## Business Recommendations
- River proximity significantly impacts housing prices.
- Premium properties represent approximately 21% of the market.
- Investors should consider river-adjacent properties for higher-value investments.
